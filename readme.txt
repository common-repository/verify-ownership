An easy way to add Google, Yahoo, Bing, Alexa, Pinterest and Norton Safe Web meta tag verification codes to your site and gets ownership verified.

== Description ==
**Verify Ownership** is a lightweight plugin that allows you to easily add your meta tag verification codes to your site. It also adds statistics scripts (e.g. Google Analytics) to the footer of your site.

This plugin has been developed to meet the new WordPress guidelines: everything that falls outside the theme design, layout and structure and adds extra functionality to your setup, needs to be separated from the theme and developed as a plugin.

You don\'t need to use up to six plugins for all these services. All in one is better!

== Installation ==
1. Download the plugin (.zip file) on your hard drive.
2. Unzip the zip file contents.
3. Upload the `verify-ownership` folder to the `/wp-content/plugins/` directory.
4. Activate the plugin through the \'Plugins\' menu in WordPress.
5. A new sub menu item `Verify Ownership` will appear in your main Settings menu.
